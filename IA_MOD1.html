Conceptos Básicos de Inteligencia Artificial

🧠 ¿Qué es la Inteligencia Artificial?

La IA es un campo de la informática que desarrolla sistemas capaces de realizar tareas que requieren inteligencia humana: aprendizaje, razonamiento, percepción, comprensión del lenguaje y resolución de problemas.

📊 Evolución de la IA

EraHito
1950-1970IA simbólica, primeros programas de ajedrez
1980-2000Sistemas expertos, machine learning temprano
2010-2020Deep learning, AlexNet, AlphaGo
2022-presenteIA Generativa, LLMs (GPT, Gemini, Claude)

🧩 Tipos de IA

  • IA Débil (Narrow AI): Especializada en una tarea (ej: ChatGPT, asistentes, recomendadores)
  • IA Fuerte (General AI): Habilidad cognitiva humana general (teórico, aún no existe)
  • Superinteligencia: Supera capacidad humana (concepto futurista)

Ingeniería de Prompts: Fundamentos

❓ ¿Qué es un prompt?

Un prompt es la instrucción que le damos a un modelo de IA para obtener una respuesta específica. La calidad del prompt determina la calidad de la respuesta.

🧱 Elementos de un buen prompt

  • Contexto: Antecedentes necesarios
  • Instrucción clara: Qué queremos que haga
  • Formato deseado: Cómo queremos la respuesta
  • Ejemplos (few-shot): Demostraciones del resultado esperado
  • Restricciones: Límites (longitud, tono, exclusiones)

⚡ Comparación: Prompt básico vs avanzado

# ❌ Básico (poco efectivo)
"Resume este texto: [texto]"

# ✅ Avanzado (efectivo)
"Actúa como un analista de datos senior. Resume el siguiente informe en 3 párrafos:
- Primer párrafo: hallazgos principales
- Segundo párrafo: implicaciones para negocio
- Tercer párrafo: recomendaciones concretas
Usa lenguaje claro, evita jerga técnica. Máximo 300 palabras.
[texto]”

🎭 Asignación de roles (Role Prompting)

Ingeniería de Prompts: Técnicas Avanzadas

🔗 Prompt Chaining (Encadenamiento)

Dividir una tarea compleja en una secuencia de prompts donde la salida de uno alimenta al siguiente.

# Paso 1: Extraer información clave
"Extrae de este artículo: (1) problema principal, (2) metodología, (3) conclusiones."

# Paso 2: Analizar con la información extraída
"Basado en [problema] y [metodología], ¿qué limitaciones tiene este estudio?"

# Paso 3: Generar recomendaciones
"Con las limitaciones identificadas, propón 3 mejoras para futuras investigaciones."

📚 Few-Shot Prompting

Proporcionar ejemplos para guiar el formato y estilo de la respuesta.

"Clasifica el sentimiento de los siguientes comentarios como Positivo, Negativo o Neutral.

Ejemplo 1: 'Me encantó el producto, funciona perfecto' → Positivo
Ejemplo 2: 'Llegó tarde y roto' → Negativo
Ejemplo 3: 'El producto es regular' → Neutral

Comentario: 'La atención al cliente fue excelente, pero el envío demoró mucho' →"

🔄 Chain of Thought (Cadena de Pensamiento)

Pedir a la IA que muestre el razonamiento paso a paso antes de dar la respuesta final.

"Resuelve el siguiente problema paso a paso, explicando tu razonamiento antes de dar la respuesta final:
'Un cliente compró 3 productos a $25 cada uno, pero tenía un cupón de 15% de descuento. También pagó $8 de envío. ¿Cuánto pagó en total?'"

# La IA mostrará: cálculo subtotal → descuento → + envío → total

🧪 Parámetros avanzados

  • Temperatura: Controla creatividad/aleatoriedad (0 = determinista, 1 = muy creativo)
  • Top_p (nucleus sampling): Selecciona tokens con probabilidad acumulada p
  • Max tokens: Límite de longitud de respuesta

Integración y Automatización de Procesos

🔌 APIs de IA: Conectando herramientas

La mayoría de las herramientas ofrecen APIs para integrar IA en tus propios flujos.

# Ejemplo: Llamada a API de ChatGPT (conceptual)
import openai

# Configurar API key
openai.api_key = "tu-api-key"

# Hacer una solicitud de completado
respuesta = openai.ChatCompletion.create(
  model="gpt-4",
  messages=[
    {"role": "system", "content": "Eres un asistente útil."},
    {"role": "user", "content": "Resume este texto: [texto]"}
  ],
  temperature=0.3 # Bajo para respuestas más precisas
)

# Extraer respuesta
print(respuesta.choices[0].message.content)

🤖 Zapier / Make (integradores low-code)

  • Zapier: Conecta apps sin código. Plan gratuito con 100 tareas/mes
  • Make (ex Integromat): Visual, más flexible. Plan gratuito con 1000 ops/mes

Proyecto Final y Mejores Prácticas

✅ Mejores prácticas en el uso de IA

  • Verificación humana: La IA puede alucinar. Siempre verifica hechos.
  • Privacidad: No compartas información sensible en versiones gratuitas.
  • Iteración: Los buenos prompts se construyen probando y refinando.
  • Documentación: Guarda prompts exitosos para reutilizarlos.
  • Ética: Usa IA como apoyo, no para suplantar tu criterio.
